# Simple Power Plan Manager

A small Windows desktop app by **@FPSHEAVEN.com** for managing Windows power plans.
<img width="980" height="669" alt="image" src="https://github.com/user-attachments/assets/c79896ea-5bd7-49c8-8d68-ebc6ac057f16" />



## Features

- View all installed power plans
- See which power plan is active
- Import and export `.pow` power plans
- Activate, duplicate, rename, edit descriptions, and delete plans
- Restore missing default Windows power plans
- Open Windows advanced power settings
- Quick link to the FPSHEAVEN YouTube channel
